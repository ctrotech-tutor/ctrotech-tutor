# ctrotech

I write TypeScript libraries and build full-stack apps.

A few things I've shipped:

- **ctrovalidate** - form validation library with bindings for React, Vue, Svelte, and Next.js. Core engine has 22 rules, async support, and i18n.
- **ctrodb** - client-side database for the browser and Node.js. IndexedDB persistence, full-text search, MongoDB-like API. Around 190 tests.
- **ctroenv** - environment variable toolkit with type-safe schemas and a CLI.
- **ctrodoc** - documentation framework for TypeScript libraries. Drop it in, write MDX, get search and API references.
- **vjsRouter** - file-system router for vanilla JavaScript SPAs. Generates route manifests from a pages directory.

I also work on contract projects. Most recently a PDF-generation platform (Next.js, Drizzle, WebSockets, Playwright), a real-time chat app, and an e-commerce admin system.

Less active on here than I should be - most of my work is in private repos or on npm.
